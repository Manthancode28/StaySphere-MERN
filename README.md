# StaySphere 🏡

**StaySphere** is a full-stack property rental web application inspired by Airbnb. It enables users to explore, list, and book short-term rental properties such as PGs, hostels, or private rooms across metro cities. The app is built with the MERN stack and features a clean, responsive UI, filtering options, and dynamic listings.

## 🌟 Key Features

- 🏘️ Browse rental listings with detailed information and photos
- 🧹 Filter properties by categories 
- 📆 View property availability and pricing (with tax toggle)
- ✍️ Property listing creation for hosts (WIP)
- 🖼️ Modern listing cards – vertical, horizontal, banner-style
- 🔍 Responsive and intuitive UI

## Live : https://staysphere-4t14.onrender.com/listings
